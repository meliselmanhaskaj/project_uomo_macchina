# Buoni Digitali Esercenti Convenzionati

Sistema web completo per la gestione di convenzioni aziendali con esercenti. 

- Molte società erogano come benefit ai clienti delle convenzioni con esercenti del territorio
- La convenzione prevede un vantaggio per il dipendente e una remunerazione nei confronti dell’esercente da parte dell’azienda
- Il sistema dovrà certificare in modo efficiente l’utilizzo della convenzione nel rispetto delle regole stabilite
